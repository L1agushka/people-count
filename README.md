# Classroom Crowd Counting System

Система подсчёта студентов в аудиториях с автоматическим вычитанием преподавателей и посторонних наблюдателей.  
Разработана для хакатона.  
**Работает полностью на CPU — видеокарта не требуется!**


## Ключевые особенности

- **Гибридный подход**: YOLOv8 (детекция) + DM-Count (плотностная карта)
- **Автоматическое вычитание "лишних"** — стоячих преподавателей и наблюдателей
- **Полная поддержка CPU** — запускается даже на ноутбуке без видеокарты
- **Бенчмарк производительности**: замер скорости обработки
- **Готово к деплою**: не требует сложной настройки

## Требования

- **ОС**: Windows / Linux / macOS
- **Python**: 3.8 или новее
- **RAM**: минимум 4 ГБ (рекомендуется 8+ ГБ)
- **Видеокарта**: **не требуется**  
  (GPU ускорит работу, но не обязателен)
